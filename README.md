# 🛠️ Super Menu de Ferramentas em C

Este é um projeto desenvolvido durante a primeira semana do curso de **Ciência da Computação**. O objetivo é consolidar conhecimentos básicos de lógica de programação, estruturas de repetição e controle de fluxo.

## 🚀 Funcionalidades
O programa centraliza diversas ferramentas úteis em um único menu interativo:
1. **Média Acadêmica**: Calcula a média aritmética simples entre duas notas.
2. **Conversor de Medidas**: Transforma metros em centímetros, milímetros ou quilômetros.
3. **Calculadora de IMC** *(Em breve)*: Calcula o Índice de Massa Corporal e fornece a classificação.

## 💻 Conceitos Aplicados
- Estruturas de repetição (`do-while` e `while`).
- Seleção múltipla (`switch-case`).
- Operadores aritméticos e lógicos.
- Entrada e saída de dados (`scanf` e `printf`).

## 🛠️ Como rodar o projeto
Para compilar e executar o código, utilize um compilador GCC:
```bash
gcc super_menu.c -o super_menu
./super_menu
